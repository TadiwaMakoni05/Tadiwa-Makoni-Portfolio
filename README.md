# Tadiwa Makoni - Portfolio

A modern, responsive, and visually appealing personal portfolio website built to showcase my projects, skills, and services as a Software Developer.

## 🚀 Description

This portfolio bridges the gap between design and engineering. Built with [Next.js](https://nextjs.org/) and styled using [Tailwind CSS](https://tailwindcss.com/) and [Framer Motion](https://www.framer.com/motion/), it offers a sleek, interactive, and highly functional user experience. The portfolio highlights my expertise across frontend and backend development, mobile apps, and UI/UX design.

## ✨ Features

- **Modern & Minimalist Design**: Clean aesthetics focusing on content readability and visual hierarchy.
- **Responsive Layout**: Fully responsive design that works seamlessly across desktop, tablet, and mobile devices.
- **Interactive Animations**: Smooth scroll effects and component transitions using Framer Motion (`FadeIn`).
- **Projects Showcase**: Detailed project cards with modal views, extended descriptions, tech stacks, and links to live demos and source code.
- **Services Section**: Comprehensive list of services offered, including System Development, AI Engineering, Mobile Apps, and UI/UX Design.
- **Tech Stack Overview**: Organized view of technologies frequently used across Frontend, Backend, Mobile, Database, and Tools categories.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (React 19)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TadiwaMakoni05/Tadiwa-Makoni-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   # or yarn install / pnpm install / bun install
   ```
4. Start the development server:

   ```bash
   npm run dev
   # or yarn dev / pnpm dev / bun dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `app/`: Next.js App Router configuration (`page.tsx`, layout, globals).
- `components/`: Modular UI sections including:
  - `hero.tsx`
  - `about.tsx`
  - `services.tsx`
  - `tech-stack.tsx`
  - `projects.tsx`
  - `contact.tsx`
- `components/ui/`: Reusable, generic UI components like `fade-in.tsx`.
- `public/`: Static assets such as images and project thumbnails.
